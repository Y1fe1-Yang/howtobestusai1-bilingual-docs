📖 What is Skills

Skills is the "ability plug-in" of HappyCapy, which is like installing different professional skills for AI. They're not just simple text instructions, they can:

  • Call external API: Connect to GitHub, Notion, Google and other third-party services to achieve cross-platform data synchronization
  • Run Python/JavaScript scripts: Perform complex data processing, image generation, video editing and other tasks
  • Use MCP protocol(Model Context Protocol): Combining various tool capabilities like Lego bricks

To put it simply: a Skill can transform HappyCapy from "good at chatting" to "good at working".

for example:

  • Install "Frontend Slides" Skill → Create richly animated HTML presentations, support conversion from PPT or build from scratch, with interactive effects and modern design
  • Install the "3D Web Experience" Skill → Build an immersive 3D web experience, including product configurator, 3D portfolio, and interactive scenes, using Three.js and WebGL technologies
  • Install the "Redbook Creator & Publisher" Skill → Create Xiaohongshu style posts (title + text + tags), generate beautiful pictures, and quickly complete content creation

Using Skills is super easy, HappyCapy will be automatically called according to your needs.

Method 1: Natural language (recommended)

Just say what you want to do, and HappyCapy will automatically select the appropriate Skill.

🔍 Search case: How to find Skills

Skills is open source, and developers around the world are contributing various practical skills. You can find it in these places:

Method 1: Ask HappyCapy directly (easiest)

Help me find a data-processing Skill

HappyCapy will use the built-infind-skillsFunction, automatically searches the open source community and recommends the most suitable Skills to you. You can choose the one you like most to install.

Method 2: Search on the community platform

  • GitHub：Search for "Skills", there are thousands of open source skills
  • SkillsMP：Skills Marketplace, browse skills by category
  • SkillsSH：Community sharing platform to discover the creative skills of other users

🛠️ Create Case: How to Skill

Creating a Skill is as simple as describing the functionality you want in natural language.

Complete Case: Job Resume Optimizer

Pain points:I submitted 100 resumes but got no response and I don’t know how to change them.

solve:Create a Skill dedicated to optimizing your resume

Help me create a Skill:

Name: Resume Optimizer

Function:
Input: My resume + target job JD

Process:
1. Analyze JD keywords and requirements
2. Compare with the resume and identify matches and gaps
3. Optimize wording to better fit the role
4. Quantify achievements with data
5. Improve layout to highlight key points
6. Generate 3 versions: conservative / balanced / aggressive

Output:
- 3 optimized resumes
- Usage guidance for each version
- Interview preparation points

And help me test it!
