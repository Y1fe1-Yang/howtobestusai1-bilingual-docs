How non-technical people use HappyCapy

Applicable people

Finance, legal, marketing, operations, product managers and other professionals without programming background

core scene

📊 Scenario 1: Automated data workflow

Real case: The finance team uses plain text to describe the workflow and let AI automatically perform complex data processing tasks.

How to do it in HappyCapy:

1. Describe your workflow in natural language

"At the beginning of every month, please help me:
1. Read last month’s sales data from Excel table
2. Calculate the growth rate for each region
3. Generate comparison chart
4. Create a report with key metrics"

2. HappyCapy will actively ask for required input

  • "Please provide Excel file"
  • "Which month's data need to be compared?"

3. Save as automated process (Pro/Max plan)

  • Set up scheduled tasks to be executed automatically every month
  • No need to re-describe each time

Key advantages

  • ✅ No need to learn Excel formulas or programming
  • ✅Describe once, use again and again
  • ✅ Reduce human errors

💡 Advice for non-technical people

1. Start with simple tasks to build confidence

  • Try first: "Generate a poster for me"
  • Then try: "Analyze this data and make a graph"
  • Last try: complex multi-step workflows

2. Describe your needs like chatting with an assistant

  • ❌ Don’t think “what tool should I use?”
  • ✅ Just say "what result do I want"
  • HappyCapy will automatically select the appropriate tools and models

3. Save successful workflow

  • Record effective prompt words
  • For repeated tasks, you can directly copy the last description
  • Pro/Max users can set up automation

4. Don’t be afraid of “dumb” questions

  • HappyCapy won’t judge you
  • Trial and error is part of learning
  • Each conversation is independent and can be restarted at any time
