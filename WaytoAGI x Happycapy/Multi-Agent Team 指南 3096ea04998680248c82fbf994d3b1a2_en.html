What is Multi-Agent Team

In complex projects, a single AI may not handle all aspects perfectly. Multi-Agent Team allows you to create specialized AI agents, each responsible for different tasks:

  • Researcher: Responsible for searching and organizing information
  • Designer: Responsible for visual design and creativity
  • Developer: Responsible for writing and optimizing code
  • Tester: Responsible for quality inspection and testing

How to use Multi-Agent Team

⚙️Enable Agent Teams function

Before using Multi-Agent Team, you need to enable this experimental feature:

  1. Click on the avatar in the lower left corner,Open「Settings」page
  1. Find the "Experimental" experimental featureOptions
  1. Turn on the "Agent Teams" switch

Just mention in the conversation that you need teamwork, and HappyCapy will automatically create and manage an Agent Team:

I need to build a complete website, including frontend, backend, and database design.
Please assemble a team to complete this project.

HappyCapy will automatically:

  1. Create the appropriate Agent role
  1. Assign tasks to each Agent
  1. Coordinate collaboration among agents
  1. Integrate all results

Applicable scenarios

  • Complex project development (requires multiple areas of expertise)
  • Content Creation (Research + Writing + Design)
  • Data analysis project (collection + cleaning + analysis + visualization)
  • Marketing activities (planning + design + copywriting + execution)

🎯 Practical case: The magic of teamwork

🍅Case: Pomodoro application development

Execution time: 3-4 minutes | Difficulty: ⭐⭐⭐⭐

Originally required: requirements analysis + UI design + front-end development + testing = a few days of work

Now: one conversation, the team automatically divides the work, and the complete application is delivered in 4 minutes

Please assemble a development team to help me build a Pomodoro app:

Product requirements:
- 25-minute work timer (pause/reset supported)
- 5-minute break reminder (sound + visual cue)
- Count completed Pomodoros for today
- Clean and modern UI design (gradients, modern style)
- Responsive layout (mobile and desktop)
- Local storage (data persists after refresh)

Suggested team roles:
1. Product Manager: define feature logic and user flows
2. UI Designer: design UI and interaction plan
3. Frontend Engineer: implement features (HTML/CSS/JavaScript)
4. QA Engineer: verify functional completeness

Please complete this collaboratively and deliver a ready-to-use web app!

Shocking point: You will get a complete web application that can be used directly, including complete code, beautiful interface, smooth animation, and reliable functions.

💡 More flexible: Different Agents in the team can also use different large models (such as Sonnet, Gemini, Haiku) to allow each agent to perform optimally in its respective areas of expertise.
