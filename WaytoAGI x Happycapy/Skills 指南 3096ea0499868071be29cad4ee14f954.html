📖 什么是 Skills

Skills 是 HappyCapy 的「能力插件」，就像给 AI 安装不同的专业技能。它们不仅仅是简单的文本指令，它们还可以：

  • 调用外部 API：连接 GitHub、Notion、Google 等第三方服务，实现跨平台数据同步
  • 运行 Python/JavaScript 脚本：执行复杂的数据处理、图像生成、视频编辑等任务
  • 使用 MCP 协议（Model Context Protocol）：像乐高积木一样组合各种工具能力

简单来说：一个 Skill 可以让 HappyCapy 从「会聊天」变成「会干活」的全能助手。

比如：

  • 安装「Frontend Slides」Skill → 创建动画丰富的 HTML 演示文稿，支持从 PPT 转换或从零构建，带有交互效果和现代设计
  • 安装「3D Web Experience」Skill → 构建沉浸式 3D 网页体验，包含产品配置器、3D 作品集、交互场景，使用 Three.js 和 WebGL 技术
  • 安装「Redbook Creator & Publisher」Skill → 创作小红书风格帖子（标题+正文+标签），生成精美配图，快速完成内容创作

🛠️ 创建案例：怎么做 Skill

创建 Skill 很简单，就是用自然语言描述你想要的功能。

完整案例：求职简历优化器

痛点： 投了 100 份简历没回应，不知道怎么改

解决： 创建一个专门优化简历的 Skill

帮我创建一个 Skill：

名称：求职简历优化器

功能：
输入：我的简历 + 目标岗位 JD

处理：
1. 分析 JD 的关键词和要求
2. 对比简历，找出匹配点和不足
3. 优化措辞（更符合岗位）
4. 量化成果（用数据说话）
5. 调整排版（突出重点）
6. 生成 3 个版本：保守版/平衡版/激进版

输出：
- 3 个优化后的简历
- 每个版本的使用建议
- 面试准备要点

并帮我测试一下！
